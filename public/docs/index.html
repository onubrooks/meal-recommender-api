    Introduction

    A fast and simple API that supplies meal recommendations for users with allergies. The API supports meals, items, users and allergies.

    This documentation aims to provide all the information you need to work with our API.

    As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

    Base URL

    https://meal-recommender-api.herokuapp.com

    Authenticating requests

    This API is not authenticated.

    Data Management

    APIs for managing data CRUD operations

    Add/Create/Update a Meal

    Create or update a meal record, supplying the name and description to be saved.

    Example request:
    curl --request POST \
    "https://meal-recommender-api.herokuapp.com/api/meals" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"Grilled Fish with Spice Chilli\",
    \"description\": \"This is the description\"
}"

    const url = new URL(
    "https://meal-recommender-api.herokuapp.com/api/meals"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "Grilled Fish with Spice Chilli",
    "description": "This is the description"
}

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
